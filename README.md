# Team Profile Generator

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description

This application will take in information about employees, then generates an HTML webpage that displays summaries for each person including different information depending on the title of the person
